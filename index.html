['메뉴 추천' 페이지 사용 방법]

메뉴 추천 페이지는 옵션에 기반해 랜덤으로 메뉴를 추천합니다.
옵션은 위치에 따라, 메뉴 구분에 따라 각 1개 이상 선택해야 합니다.
옵션을 변경하고 싶다면, '옵션 설정' 버튼을 눌러주세요.
추천 받은 메뉴가 마음에 들지 않는다면, 다시 한 번 추천 받기 버튼을 눌러주세요.

['근처 식당 지도 보기' 페이지 사용 방법]

근처 식당 안내는 선택한 문 근처의 선택한 메뉴를 파는 식당을 모두 보여줍니다.
페이지에 표시되는 식당의 개수가 너무 많을 수 있으므로,
옵션(학교 문, 음식 분류)은 각 1개씩만 선택 가능합니다.
옵션을 변경하고 싶다면, '옵션 설정' 버튼을 눌러주세요.
